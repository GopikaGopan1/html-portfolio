It showcases my front-end projects and demonstrates the use of different HTML elements and page structures.

- Built completely with HTML5 (no CSS or JavaScript)
- Demonstrates proper use of headings, lists, links, and images
- Includes multiple project pages:

  - Birthday Invite Project
  - About Me
